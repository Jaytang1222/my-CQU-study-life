# 背景

在计算机科学与技术领域，**概率论与数理统计**在系统可靠性分析和模型评估中至关重要。例如，在评估硬件或软件系统的**寿命**时，**指数分布**（Exponential Distribution）常被用来建模系统无故障运行的时间（Time To Failure）。指数分布的概率密度函数（PDF）为 $f(t; \lambda) = \lambda e^{-\lambda t}$，其中 $\lambda$ 是**故障率**（rate parameter）。

# 问题

某批次计算机服务器的无故障运行时间 $T$（单位：年）被认为服从指数分布，其唯一的参数是故障率 $\lambda$。

为了估计这批服务器的故障率 $\lambda$，工程师随机监测了 $n=5$ 台服务器，记录到的无故障运行时间 $T_i$（单位：年）数据如下：
$$T_1=2.0, T_2=3.5, T_3=1.5, T_4=2.5, T_5=3.0$$
请利用**矩估计法**（Method of Moments, MM），根据样本均值等于总体期望的原理，求出该服务器批次故障率 $\lambda$ 的**矩估计值 $\hat{\lambda}$**。

# 解决
1.建立估计方程

对于服从指数分布 $T \sim \text{Exp}(\lambda)$ 的随机变量，其总体期望为：
$$E[T] = \frac{1}{\lambda}$$

样本均值 $\overline{T}$ 的计算公式为：
$$\overline{T} = \frac{1}{n} \sum_{i=1}^{n} T_i$$

因此，矩估计方程为：
$$\frac{1}{\hat{\lambda}} = \overline{T}$$

解得 $\lambda$ 的矩估计值 $\hat{\lambda}$：
$$\hat{\lambda} = \frac{1}{\overline{T}}$$

2.计算样本均值 $\overline{T}$

首先计算样本数据的总和：
$$\sum_{i=1}^{5} T_i = 2.0 + 3.5 + 1.5 + 2.5 + 3.0 = 12.5$$

计算样本均值 $\overline{T}$：
$$\overline{T} = \frac{12.5}{5} = 2.5 \text{ 年}$$

3.计算 $\lambda$ 的矩估计值 $\hat{\lambda}$

将 $\overline{T}$ 代入 $\hat{\lambda} = \frac{1}{\overline{T}}$：
$$\hat{\lambda} = \frac{1}{2.5} = 0.4$$

**结论：** 该批次服务器的故障率 $\lambda$ 的矩估计值 $\hat{\lambda}$ 为 $\mathbf{0.4}$ 次/年。
